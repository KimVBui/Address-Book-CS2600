# Address Book - CS2600 Project

Compiling the code can be done using the command(s):
```bash
gcc src/*.c -o AddressBook

gcc src/*.c -o AddressBook.exe
```

Compiling the tests for the code can be done using the command(s):
```bash
gcc tests/*.c src/address_book_fops.c src/address_book_menu.c -o AddressBookTests

gcc tests/*.c src/address_book_fops.c src/address_book_menu.c -o AddressBookTests.exe
```
